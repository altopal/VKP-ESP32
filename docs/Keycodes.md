# Keypad key codes
## Numbering

* The first 4 bits are the row
* The second 4 bits are the column

## Codes

    1 = 0x01
    2 = 0x02
    3 = 0x04
    4 = 0x11
    5 = 0x12
    6 = 0x14
    Play/Quit = 0x18
    7 = 0x21
    8 = 0x22
    9 = 0x24
    Rec/Yes = 0x28
    * = 0x31
    0 = 0x32
    # = 0x34
    Light/No = 0x38
